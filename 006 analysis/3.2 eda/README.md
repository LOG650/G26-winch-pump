# 3.2 Eksplorativ dataanalyse (EDA)

**WBS-aktivitet:** 3.2
**Eier:** Tord
**Status:** Ikke startet

## Formål

Visualisere data, identifisere mønstre og analysere datastruktur som grunnlag for gap-deteksjonen.

## Leveranser

- Python-skript for EDA
- Visualiseringer: tidsserier, korrelasjoner, sesongmønstre
- Mønsteridentifikasjon og datastrukturanalyse
- Deskriptiv statistikk dokumentert

## Filnavn-konvensjon

- `fig_*.png` – figurer brukt i rapporten
- `tab_*.md` – tabeller brukt i rapporten

## Akseptansekriterier

Deskriptiv statistikk og visualiseringer dokumentert og reproduserbare.
