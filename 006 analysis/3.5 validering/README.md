# 3.5 Validering og testing

**WBS-aktivitet:** 3.5
**Eier:** Begge
**Status:** Ikke startet
**Milepæl:** M6 – varslingssystem testet og validert mot kjente gap (2026-05-10)

## Formål

Teste systemet mot kjente gap, verifisere korrekt varsling og teste edge cases.

## Leveranser

- Testrapport mot kjente kapasitetsgap i historiske data
- Verifisering av korrekt deteksjon og varsling
- Edge case-testing (ingen gap, flere samtidige gap, forverring over tid)

## Akseptansekriterier

Systemet identifiserer gap korrekt, sender varsling til riktig mottaker og håndterer edge cases.
